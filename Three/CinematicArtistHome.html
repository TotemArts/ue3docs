Cinematic Artist

Cinematic sequences have many uses when it comes to video games. They can be used to create scripted gameplay sequences, used as cutscenes in-game, used as promotional content for marketing, etc. Unreal Engine 3 provides tools, in the formof Matinee, Kismet, post process effects, etc., that make it possible to create extremely high-quality, professional, and stylized cinematic sequences using the same realtime rendering pipeline the game uses for standard gameplay. Knowing how to use these tools can make the job of the cinematic artist enjoyable and result in amazing cinematic experiences for the player or consumer.
